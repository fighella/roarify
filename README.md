# ROARify

Access to the Shopify API ( https://docs.shopify.com/api ) using the amazing ROAR ( https://github.com/apotonick/roar )

## Installation

Add this line to your application's Gemfile:

    gem 'roarify'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roarify
